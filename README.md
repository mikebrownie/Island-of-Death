# Island-of-Death
A text-based game I made in C++ as my Introduction to Programming project.
To start, download the files and execute the a.out executable into a terminal in the downloaded directory (Try typing ./a.out)

## Gameplay

### Introduction
![Image of introduction](https://mikebrownie.github.io/uploads/island-of-death/intro.PNG)
Just a little story to the game.

### Main Menu
![Image of main menu](https://mikebrownie.github.io/uploads/island-of-death/main_menu.PNG)
This main menu shows:
- A map including:
  - The player posistion
  - The cannon posistion (main objective)
  - Obstacles
- Possible actions the player may take:
  - Move and pass a day
  - Rest and pass a day
  - Check inventory, where player can use an item
- The date

### Inventory
![Image of inventory](https://mikebrownie.github.io/uploads/island-of-death/inventory.PNG)
Items include:
- Medkits for healing
- Bullets for fighting
- Invisibility potions for avoiding misfortunes
- Gold for buying more items

### Misfortunes
![Image of misfortune](https://mikebrownie.github.io/uploads/island-of-death/misfortunes.PNG)
Misfortunes are intended to add a layer of difficulty to the game. Misfortunes are randomly selected each day.

### Shop
![Image of shop](https://mikebrownie.github.io/uploads/island-of-death/shop.PNG)
The shop is entered when the player befriends an islander. The player can buy items here.
